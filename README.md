# Бегущая строка на Django
## Зависимости
##### Для работы приложения потребуется установить несколько библиотек:
##### Django:
```bash
pip install django
```
##### Moviepy:
```bash
pip install moviepy
```
##### Imagemagic 
На windows может потребоваться ручная установка (https://imagemagick.org/script/download.php)
